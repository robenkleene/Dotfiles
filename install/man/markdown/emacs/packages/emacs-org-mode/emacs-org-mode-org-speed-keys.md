- Enabled with `(setq org-use-speed-commands t)`
- Use `M-m` to move the cursor to the correct spot on the line

# Speed Keys

## Navigation

- `p` / `n`: Next / previous item
- `b` / `f`: Next / previous sibling

## Deleting

- `k`: Cut subtree

## Moving

- `l` / `r`: Shift left / right
- `w`: Refile

## Todo

- `t`: Todo
