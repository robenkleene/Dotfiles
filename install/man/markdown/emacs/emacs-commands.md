# `emacs` Commands

- `C-x C-c`: Quit
- `C-x d` or `C-x C-j`: Enter dired mode, second version jumps to current file
- `C-c`: Bindings reserved for the user
- `C-x z`: Repeat last command
- `M-:`: Evaluate lisp

## Universal Argument

- `C-u 2` or `M-2`: Perform an action passing `2` as the argument
