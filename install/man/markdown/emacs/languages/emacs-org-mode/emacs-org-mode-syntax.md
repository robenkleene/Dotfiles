# Basic

- `*bold*`
- `/italic/`
- `_underlined_`
- `=verbatim=`
- `~code~`

# Document

Specifying a title:

```
#+title: Todo
```

Block quote:

```
> Quote
```

# Drawer

Drawers are useful for easily collapsible notes that aren't source code

- `C-c C-x d`: Insert drawer (it will prompt for a drawer name)

```
:<drawer name>:
Drawer text
:end:
```
