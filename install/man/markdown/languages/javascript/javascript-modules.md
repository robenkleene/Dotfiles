# Modules

CommonJS vs. ES6 modules. CommonJS 

## CommonJS

### Import

``` javascript
const package = require('module-name')
```

### Export

``` javascript
module.exports = value
```

## ES6 Modules

### Import

``` JavaScript
import randomSquare from './modules/square.js';
```

### Export

``` JavaScript
export default randomSquare;
```
