# Regular Expressions

These need to be escaped outside of character classes:

	\.[]{}()<>*+-=!?^$|

These need to be escaped inside of character classes:

	^-]\

## Perl Character Classes

- `\d`: Digit
- `\D`: Not digit
- `\s`: Whitespace
- `\S`: Not whitespace
- `\w`: Word character
- `\b`: Not word character
- `\W`: Not word character
