# Remote

- `cd`
- `ls`
- `pwd`
- `rm`: Delete remote file (no local equivalent)
- `get <filename>`: download file (doesn't support tab completion)

# Local

- `lcd`
- `lls`
- `lpwd`
- `put <filename>`: Upload file (doesn't support tab completion)

## More

- Prefix any command with `!` to run it locally, e.g., `!ls`
