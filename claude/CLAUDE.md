# Abstraction and Design

Before writing new code, ask: "Does an abstraction already exist or want to exist here?" If you're about to duplicate logic, write a shared helper first. Prefer small, focused classes/functions over complex interdependencies. Default to the simplest abstraction that eliminates duplication.
